Ранги проверок в системе Phoenix используются для определения сложности задач, превышающих человеческие возможности:

- **Второй ранг**: Сложности, соответствующие действиям сверхчеловеческих существ.
- **Третий ранг**: Для полубожественных сущностей.
- **Четвертый ранг**: Для божественных аспектов
- **Пятый ранг**: Для отражения полной божественной силы.
- **Увеличение сложности**: Каждый ранг повышает сложность проверки в два раза.
